# My U Library - backend app

Frontend app for My U Library, coded with NodeJs + Express

## Run Locally

Clone the project

```bash
  git clone https://github.com/angelH98-sys/my-u-library-backend.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
npm install
```

Start the server

```bash
node start index
```
